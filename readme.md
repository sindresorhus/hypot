# hypot

> ES2015 [`Math.hypot()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/hypot) [ponyfill](https://ponyfill.com)


## Install

```
$ npm install hypot
```


## Usage

```js
const hypot = require('hypot');

hypot(3, -4);
//=> 5
```
